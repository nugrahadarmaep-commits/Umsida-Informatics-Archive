# 🤖 Bot Auto Nikko Puzzle

Repository ini berisi skrip Python sederhana untuk menyelesaikan tugas "Fun Game" (Nikko Puzzle) secara otomatis.

## ⚙️ Cara Kerja Bot (Secara Singkat)
1. Menjalankan file `.exe` game menggunakan library bawaan Python (`subprocess`).
2. Bot membaca teks pertanyaan yang muncul dari game.
3. Mencari angka di dalam teks tersebut (menggunakan `regex`).
4. Menghitung otomatis luas/keliling/volume sesuai bangun ruang yang ditanyakan.
5. Mengirimkan jawaban kembali ke dalam game secara otomatis.

## 🎮 Cara Menjalankan
1. Pastikan file executable game berada dalam satu folder yang sama dengan `bot.py`.
2. Jalankan perintah di terminal: `python bot.py`
